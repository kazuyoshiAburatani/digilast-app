DIGILAST

利用規約

最終更新日：2026年4月21日

本利用規約（以下「本規約」）は、油谷和好（以下「当方」）が提供するデジタルデータ管理サービス「DIGILAST」（以下「本サービス」）の利用条件を定めるものです。本サービスをご利用いただくにあたり、本規約に同意いただいたものとみなします。

第1条（定義）

本規約において使用する用語の定義は以下の通りです。

  1. 「利用者」とは、本サービスに登録し利用する個人をいいます。
  2. 「暗号化パスワード」とは、利用者が設定する、データの暗号化および復号に使用するパスワードをいいます。
  3. 「チェックイン」とは、利用者が本サービスに定期的にアクセスし、生存確認を行う操作をいいます。
  4. 「トリガー」とは、チェックイン期限を超過した場合に開始される、段階的な通知および削除プロセスをいいます。

第2条（サービスの内容）

本サービスは、以下の機能を提供します。

  1. デジタルデータのクライアントサイド暗号化および保管
  2. 定期的なチェックイン（生存確認）機能
  3. チェックイン未実施時の段階的通知および自動データ削除機能
  4. その他、当方が適宜提供する関連機能

第3条（アカウント登録）

  1. 利用者は、正確かつ最新の情報を登録するものとします。
  2. 利用者は、自己のアカウント情報（メールアドレス、パスワード、暗号化パスワード）を適切に管理する責任を負います。
  3. アカウントの不正利用により生じた損害について、当方は一切の責任を負いません。

第4条（暗号化パスワードに関する重要事項）

重要：本サービスはゼロ知識暗号化を採用しており、当方は利用者の暗号化パスワードを一切保持・管理しません。暗号化パスワードを紛失された場合、保管されたデータの復号は技術的に不可能となり、データを復元することはできません。
  1. 暗号化パスワードの管理は、利用者の自己責任において行うものとします。
  2. 当方は、暗号化パスワードの紛失に起因するデータの喪失について、一切の責任を負いません。
  3. 当方は、暗号化パスワードの再発行、復旧、リセット等の対応を行うことができません。

第5条（料金および支払い）

  1. 本サービスには無料プランと有料プランがあります。各プランの内容および料金は、本サービスのウェブサイトに掲示するとおりとします。
  2. 有料プランの料金は、クレジットカード決済により支払うものとします。
  3. 支払済みの料金は、法令に定める場合を除き、返金いたしません。
  4. 当方は、事前に通知のうえ、料金を変更することがあります。

第6条（チェックインおよびデータ削除）

  1. 利用者は、設定したチェックイン間隔内に本サービスにチェックインを行う必要があります。
  2. チェックイン期限を超過した場合、当方は段階的な通知（メール通知等）を行い、所定の猶予期間経過後にデータの削除を実行します。
  3. 削除されたデータは復元できません。
  4. 通知メールの不着（迷惑メールフィルタ、メールアドレス変更等）に起因するデータ削除について、当方は責任を負いません。

第7条（禁止事項）

利用者は、本サービスの利用にあたり、以下の行為を行ってはなりません。

  1. 法令または公序良俗に違反する行為
  2. 犯罪に関連する行為、または犯罪の証拠隠滅を目的とした利用
  3. 本サービスのサーバーまたはネットワークに過度の負荷をかける行為
  4. 本サービスの運営を妨害する行為
  5. 他の利用者のアカウントを不正に使用する行為
  6. 本サービスを逆コンパイル、リバースエンジニアリングする行為
  7. その他、当方が不適切と判断する行為

第8条（サービスの変更・中断・終了）

  1. 当方は、利用者への事前通知なく、本サービスの内容を変更し、または本サービスの提供を一時的に中断することがあります。
  2. 当方は、30日前までに利用者に通知することにより、本サービスを終了することができます。
  3. 当方は、サービスの変更、中断、または終了により利用者に生じた損害について、故意または重大な過失がある場合を除き、責任を負いません。

第9条（免責事項）

  1. 当方は、本サービスが利用者の特定の目的に適合すること、期待する機能・正確性・有用性を有すること、および不具合が生じないことを保証しません。
  2. 当方は、本サービスを通じて保管されたデータの完全性、安全性、または確実な削除を保証しません。
  3. システム障害、通信障害、天災、その他当方の責に帰さない事由により、データの削除が実行されない場合、または意図しないタイミングで削除が実行された場合について、当方は責任を負いません。
  4. 本サービスの利用に関連して利用者に生じた損害について、当方の故意または重大な過失に起因する場合を除き、当方は責任を負いません。

第10条（損害賠償の制限）

  1. 当方が利用者に対して損害賠償責任を負う場合であっても、その賠償額は、当該利用者が過去12ヶ月間に当方に支払った利用料の総額を上限とします。
  2. 当方は、間接損害、特別損害、逸失利益、データの喪失に係る損害については、予見可能性の有無にかかわらず、責任を負いません。

第11条（個人情報の取扱い）

利用者の個人情報の取扱いについては、別途定める「プライバシーポリシー」に従うものとします。

第12条（知的財産権）

本サービスに関する知的財産権は、すべて当方に帰属します。利用者は、本サービスの利用により、当方の知的財産権を取得するものではありません。

第13条（利用者によるデータ）

利用者が本サービスにアップロードしたデータの権利は、利用者に帰属します。当方は、サービスの提供に必要な範囲でのみ、当該データを取り扱います。なお、データは暗号化された状態で保管されるため、当方がデータの内容にアクセスすることはありません。

第14条（退会）

  1. 利用者は、本サービスの設定画面から退会手続きを行うことができます。
  2. 退会時に、利用者のアカウントおよび保管データは削除されます。
  3. 退会後のデータ復元はできません。

第15条（規約の変更）

  1. 当方は、必要に応じて本規約を変更することがあります。
  2. 変更後の規約は、本サービスのウェブサイト上に掲示した時点で効力を生じるものとします。
  3. 重要な変更の場合は、事前にメール等で通知いたします。

第16条（準拠法・管轄裁判所）

  1. 本規約の解釈および適用は、日本法に準拠するものとします。
  2. 本サービスに関する紛争については、当方の住所地を管轄する裁判所を第一審の専属的合意管轄裁判所とします。
